Overview

This Home Assistant configuration implements real-time monitoring of HVAC performance against rigorously-established baselines derived from four years of operational analysis (2022-2025). The system applies statistical process control methodology typically used in manufacturing to residential heating systems.

All target values, statistical thresholds, and alert logic are derived from the companion Residential HVAC Performance Baseline repository, which documents the comprehensive energy analysis that established these diagnostic thresholds.

Getting Started

Prerequisites

  • Home Assistant OS or Container installation
  • Smart thermostats with Home Assistant integration
  • Weather integration (Pirate Weather recommended)
  • Basic understanding of YAML configuration

Installation Steps

  1. Clone the repository 
    git clone https://github.com/wkcollis1-eng/home-assistant-config.git
  2. Configure secrets 
    cp secrets.yaml.example secrets.yaml
# Edit secrets.yaml with your API keys
  3. Adjust baseline values in configuration.yaml to match your system
  4. Update thermostat entity IDs in configuration.yaml
  5. Install custom components via HACS: Pirate Weather
  6. Restart Home Assistant
  7. Import dashboards from dashboards/cards/

Validation

This repository uses GitHub Actions for automatic validation:

  • yamllint - YAML syntax validation
  • HA Config Check - Home Assistant configuration validation

Local validation:

# YAML syntax check
yamllint configuration.yaml automations.yaml

# Home Assistant config check (requires Docker)
docker run --rm -v "$(pwd)":/config homeassistant/home-assistant:stable \
  python -m homeassistant --script check_config --config /config
